# Lessons Learned

## Alert Fatigue Matters
Overly aggressive triggers may have looked good on a dashboard but generated excessive noise.

## Hostname Changes Create Operational Challenges
Renaming workstations caused stale hosts that needed to be cleaned up.

## API Integrations Require Consistent Data Structures
Meraki telemetry parsing depended heavily on consistent data structures to create useful dashboard widgets.

## Visualization Is Operationally Important
Raw telemetry is less useful without an easy way to interpret it at-a-glance and over time as a trend.
